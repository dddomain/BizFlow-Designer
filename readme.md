# **BizFlow Designer**

## **概要**

**BizFlow Designer** は、ブラウザひとつで動作する、軽量かつ高機能なアクティビティ図（業務フロー図）作成ツールです。

専用ソフトのインストールやユーザー登録、インターネット接続は一切不要です。配布されたHTMLファイルをブラウザで開くだけで、誰でもすぐに直感的な操作で美しい図を作成できます。エンジニアや業務担当者が、思考を止めることなくプロセスを可視化できるよう設計されています。

## **主な特徴**

* 🚀 **完全オフライン動作**  
  * 外部サーバーへの通信を行わないため、機密性の高い業務フローも安心して作成できます。セキュリティ制限の厳しい環境でも利用可能です。  
* ✨ **AI作成支援機能**  
  * 「こんなフロー図を作りたい」という内容から、ChatGPT等の生成AI用プロンプトを自動生成。AIが出力したコードを読み込むだけで、図を自動描画できます。  
* 🖱️ **直感的な操作性**  
  * ドラッグ＆ドロップでの図形配置、右クリックでのクイック追加など、マニュアル不要で使い始められます。  
* 🎨 **美しい描画と柔軟な調整**  
  * 接続線のルート調整、アイコンの配置、配色の変更など、プレゼンテーション品質の図を作成するための詳細な編集機能を備えています。  
* 📄 **多彩なエクスポート**  
  * **画像**: 背景透過PNG、JPEG  
  * **ベクター**: SVG（Illustrator等での再編集に最適）  
  * **文書**: PDF  
  * **データ**: JSON（再編集用）

## **起動方法**

特別なセットアップは不要です。

1. このリポジトリから `index.html` （または `bizflow_designer_v7_4.html`）をダウンロードします。  
2. ファイルをダブルクリックしてブラウザで開きます。  
3. すぐにアプリケーションが起動します。

## **基本的な使い方**

1. **配置**: 左側のパレットから、使いたい図形（処理、判断、アイコンなど）をキャンバスへドラッグ＆ドロップします。  
2. **接続**: 図形にマウスを乗せると現れる「●（ポート）」をクリックし、別の図形までドラッグして線を引きます。  
3. **AI活用**: ツールバーの「✨ AIを使って作成」ボタンを押し、テンプレートを使ってAIに構成案を作らせることで、作業時間を大幅に短縮できます。  
4. **保存**: ツールバー右側の「保存」メニューから、用途に合わせて形式を選んでダウンロードします。
